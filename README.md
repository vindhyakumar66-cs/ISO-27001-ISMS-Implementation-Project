# ISO 27001 ISMS Implementation Project

## Overview

This repository contains a beginner-friendly, end-to-end implementation of an Information Security Management System (ISMS) based on **ISO/IEC 27001:2022**.

The project simulates the implementation of an ISMS for a fictional SaaS company and demonstrates the key documentation, risk assessment process, security controls, and governance activities commonly performed during an ISO 27001 implementation.

The objective of this project is to showcase practical Cyber GRC knowledge and build a portfolio demonstrating the documentation and processes involved in implementing an Information Security Management System.

---

# Project Objectives

* Understand the ISO/IEC 27001:2022 implementation process.
* Perform asset identification and classification.
* Conduct information security risk assessment.
* Create a Risk Register.
* Develop a Statement of Applicability (SoA).
* Prepare key ISMS policies and documentation.
* Demonstrate internal audit and management review activities.
* Maintain sample implementation evidence.

# Documents Included

### Asset Inventory

An inventory of organizational assets including asset type, owner, classification, location, and operational status.

### Risk Register

A risk assessment identifying organizational risks, impact, likelihood, calculated risk scores, and recommended treatments.

### Statement of Applicability (SoA)

A list of applicable ISO/IEC 27001:2022 controls with implementation justifications.

### ISMS Scope

Defines the organizational boundaries and applicability of the Information Security Management System.

### Information Security Policy

High-level security policy describing management commitment toward information security.

### Access Control Policy

Defines user access management, authentication, authorization, and account management requirements.

### Incident Response Policy

Defines the process for identifying, reporting, responding to, and recovering from information security incidents.

### Internal Audit Checklist

Sample audit checklist used to verify implementation of ISMS controls.

### Management Review Report

Sample management review documenting ISMS performance and continual improvement activities.

### Evidence

Sample implementation records demonstrating how selected ISO 27001 controls may be evidenced during an audit.
